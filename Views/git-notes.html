Git Notes

Local

  • Locally means that things are happening just on my computer.
  • Local commits back up to a local repository and others can't access until we push to a remote.

Remote

  • Remote means your browser connects to an online server.
  • Remote repositories can be accessed by fellow team members(private repo) or publicly(public repo).

Local Set up

  1. Open your command line/terminal
  2. Use the cd command to navigate to your folder. For example cd desktop/playground
  3. Type in git init. THis will set up a repository(repo) for you locally.
  4. A message like this should pop up: "Initialized empy git repository in dir/filename/stuff/here"

Local Commits

  1. Make sure you're in the proper directory in the command bar.
  2. Enter git add . Note:This will add all the files you've changed.
  3. Enter git command -m "A note about the things you've changed" Note: The words in quotes can be anything.
  4. Your code is now backed up locally.
